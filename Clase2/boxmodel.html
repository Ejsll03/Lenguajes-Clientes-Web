El Modelo de Caja en CSS

En CSS, el Modelo de Caja define cómo los elementos HTML se representan como cajas rectangulares en la página. Cada caja está compuesta por varias capas: contenido, relleno (padding), borde (border) y margen (margin).

Componentes del Modelo de Caja

A continuación, se explica cada parte del Modelo de Caja con un ejemplo práctico:

Padding: Es el espacio entre el contenido y el borde de la caja. En este ejemplo, el padding es de 20px.
Border: Es la línea que rodea la caja. Aquí el borde tiene un ancho de 5px y un color azul.
Margin: Es el espacio exterior entre la caja y otros elementos. En este caso, el margen es de 20px.

Propiedad box-sizing

La propiedad box-sizing define cómo se calcula el tamaño total de la caja. Puede tomar dos valores principales:

  • content-box (por defecto): El tamaño de width y height incluye solo el contenido. El padding y el borde se suman al tamaño total.
  • border-box: El tamaño de width y height incluye el contenido, el padding y el borde. Es útil para diseños más predecibles.

En este ejemplo, se utiliza box-sizing: border-box, lo que significa que el padding y el borde están incluidos en el tamaño total de la caja.
